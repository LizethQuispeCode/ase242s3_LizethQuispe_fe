**¿Qué es Angular?**

Angular es un framework de desarrollo frontend creado por Google que permite construir aplicaciones web modernas, dinámicas y estructuradas. Está basado en el uso de componentes, lo que facilita la organización del código y el mantenimiento de la aplicación. Angular utiliza principalmente TypeScript, junto con HTML y CSS, y es muy usado para crear aplicaciones web de una sola página (SPA).

**¿Qué herramientas se necesitan instalar?**

Para desarrollar una aplicación con Angular es necesario contar con las siguientes herramientas:

-Node.js: Permite ejecutar JavaScript en el entorno local y es necesario para usar npm.

-npm (Node Package Manager): Es el gestor de paquetes que se utiliza para instalar librerías y dependencias del proyecto.

-Angular CLI: Es una herramienta de línea de comandos que facilita la creación, configuración y ejecución de proyectos Angular.
